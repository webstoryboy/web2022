01.

카드 유형 : 카드01

난이도 쉬움

소스 다운로드

완성 소스 다운받기

소개

안녕하세요!. 웹스토리보이입니다. 이번 주제는 카드 유형을 알아보겠습니다. 카드 유형은 사이트 제작에 있어 많이 사용되는 유형입니다. 사이트 제작할 때 안들어 가는 경우가 없습니다. 매번 사용하는 만큼 중요하겠죠! 일반적인 카드 유형과 웹 표준을 준수한 카드 유형을 비교해보면서 제작해보겠습니다. 그럼 준비 되셨죠? 시작하겠습니다.

이론

우리가 이번에 만들어 볼 유형은 이런 디자인입니다. 카드가 3개 나와있는 구조입니다. 많이 볼 수 있는 구조입니다. 우선 card_code01.html 페이지를 만들고, !를 치고 tab을 누르겠습니다. 언어를 ko로 변경하고 title은 카드 유형01로 변경하겠습니다. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>카드 유형01</title>
    <style>
        * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }
        .card {
            width: 100%;
            background: rgba(0,0,0,0.2);
            margin: 120px 0;
        }
        .container {
            width: 1160px;
            margin: 0 auto;
            padding: 0 20px;
            background: rgba(0,0,0,0.3);
        }
        .card-inner {
            height: 600px;
        }
    </style>
</head>
<body>
    <section class="card">
        <div class="container">
            <div class="card-inner">

            </div>
        </div>
    </section>
</body>
</html>

그럼 section을 만들고, 가운데 올 수 있도록 container를 설정합니다.

.card-inner 클래스를 만들고 자식 요소로 .cart 박스를 3개 만듭니다.

전체(*) 선택자를 이용하여 간격을 초기화하고, 각각의 영역을 잡아 가상의 색을 넣어주겠습니다. 전체 영역에는 width:100%로 설정하고 가운데 영역은 width:1160px로 잡고 가운데 올 수 있도록 margin: 0 auto를 설정하겠습니다. 간격 설정을 위한 margin 값과 padding 값을 넣어주어 간격을 설정하고, 영역 확인을 위해 가상으로 backgorund를 넣어주겠습니다. 

<section class="card-wrap section gmark">
    <div class="container">
        <h2 class="h2">웹스토리보이 강의</h2>
        <p class="desc">
            웹디자이너, 웹 퍼블리셔, 프론트앤드 개발자를 위한 강의 사이트입니다.<br>
            Gmarket Sans Light 22px 150% #67778A 
        </p>
        <div class="card-inner">

        </div>
    </div>
</section>

h2 태그와 p 태그를 만들어 주고, 내용도 작성합니다. .h2클래스와 .desc 클래스를 추가하고, 상위 박스에 section 클래스를 추가할 것입니다. 이 제목 구조는 다른 섹션에서도 똑같이 사용 될 것이기 때문에 이름을 정형화하여 재사용 할 것입니다. 

<link href="https://webfontworld.github.io/gmarket/GmarketSans.css" rel="stylesheet">

지마켓 웹 폰트를 사용하기 위해 파일을 연동하겠습니다. 1 

.gmark {
    font-family: 'GmarketSans';
}
.section {
    padding: 120px 0;
    text-align: center;
}
.section .h2 {
    font-size: 50px;
    font-weight: 500;
    margin-bottom: 20px;
    line-height: 1;
}
.section .desc {
    font-size: 22px;
    font-weight: 300;
    margin-bottom: 80px;
    line-height: 150%;
    color: #67778A;
}

지마켓 클래스를 추가하고 font-family를 설정합니다. section 클래스도 추가하고 제목과 설명 부분의 폰트와 간격을 설정합니다. 이 부분도 항상 재사용을 할 것이기 때문에 클래스 이름을 따로 표현했습니다. 재사용에 대한 부분이 이해안가도, 계속 따라하시면 나중에는 이해가 갈 것입니다. 처음에 이해가 안간다고 너무 답답해 할 필요는 없습니다. 반복적으로 사용하고 적용하다보면 자동적으로 이해가 갈 것입니다.
