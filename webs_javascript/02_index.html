자바스크립트 : 데이터 출력하기

소개

안녕하세요!. 웹스토리보이입니다. 자바스크립트를 통해 데이터 저장하는 방법을 배웠으면, 이제는 출력하는 방법 즉 불러오는 방법에 대해서 배우겠습니다.

01. 변수를 통해 데이터를 출력합니다.

변수를 저장할 때에는 하나씩 설정해서 작업 할 수도 있지만 ,(쉼표)를 써서 작업이 가능합니다. 표현방식의 차이입니다.

{
    //let x = 100;
    //let y = 200;
    //let z = "javascript";

    let x = 100, 
        y = 200, 
        z = "javascript";

    document.write(x);
    document.write(y);
    document.write(z);
}
결과 확인하기
100
200
javascript

02. 상수를 통해 데이터를 출력합니다.

상수를 통해 변수를 저장할 수 있으며, 변수를 쓰는 것보다, 상수를 써서 작업하면 오류를 줄일 가능성이 높아집니다. 그래서 변하는 값이 아니라면, 상수를 기본적으로 사용하는 것이 더 좋습니다. 

{
    const x = 100, y = 200, z = "javascript";  

    document.write(x);
    document.write(y);
    document.write(z);
}
결과 확인하기
100
200
javascript

03. 배열을 통해 데이터를 출력합니다.

배열은 여러개의 데이터가 저장되어 있습니다. 출력할 때에도 순차적으로 출력할 수 있습니다.

{
    const arr = [100, 200, "javascript"];

    document.write(arr[0]);
    document.write(arr[1]);
    document.write(arr[2]);
}
결과 확인하기
100
200
javascript

04. 2차 배열을 통해 데이터를 출력합니다.

배열 안에 배열이 있는 2차 구조입니다. 3차, 4차 구조도 가능합니다.

{
    const arr = [100, 200, ["javascript", "jquery"]];

    document.write(arr[0]);
    document.write(arr[1]);
    document.write(arr[2]);
    document.write(arr[2][0]);
    document.write(arr[2][1]);
}
결과 확인하기
100
200
javascript,jquery
javascript
jquery

05. 배열의 데이터 갯수를 출력합니다.

배열 안에 들어있는 데이터의 갯수를 구합니다. length 속성 값을 이용하여 갯수를 구합니다.

{
    const arr = [100, 200, "javascript"];

    document.write(arr.length);
}
결과 확인하기
3

06. 배열의 데이터를 for문을 통해 불러옵니다.

배열 안에 데이터가 많다면 반복적으로 실행해야 합니다. 가장 보편적으로 많이 쓰는 방법이 for문을 통해 출력하는 방법입니다.

{
    const arr = [100, 200, 300, 400, 500];

    // document.write(arr[0])
    // document.write(arr[1])
    // document.write(arr[2])
    // document.write(arr[3])
    // document.write(arr[4])

    //for(초기값, 조건식, 증감식)
    for( let i=0; i<arr.length; i++ ){
        document.write(arr[i]);
    }
}
결과 확인하기
100, 200, 300, 400, 500

07. 배열의 데이터를 forEach를 통해 불러옵니다.

배열 안에 들어있는 데이터는 forEach() 메서드를 이용하면, 더 편하게 불러올 수 있습니다. forEach 인자 값에는 요소, 인덱스, 배열 등 3가지 값을 불러 올 수 있습니다. 

{
    const num = [100, 200, 300, 400, 500];

    document.write(num[0])
    document.write(num[1])
    document.write(num[2])
    document.write(num[3])
    document.write(num[4])

    //for문
    for( let i=0; i<num.length; i++ ){
        document.write(num[i]);
    }

    //forEach()
    num.forEach(function(el){
        document.write(el);
    });

    //forEach(element, index, array)
    num.forEach(function(element, index, array){
        document.write(element);
        document.write(index);
        document.write(array);
    });
}

08. 배열의 데이터를 for of를 통해 불러옵니다.

{
    const num = [100, 200, 300, 400, 500];

    for( let i of num ){
        document.write(i);
    }
}

09. 배열의 데이터를 for in을 통해 불러옵니다.

{
    const num = [100, 200, 300, 400, 500];

    for( let i in num ){
        document.write(i);
    }

    for( let i in num ){
        document.write(num[i]);
    }
}

10. 배열의 데이터를 map()를 통해 불러옵니다.

{
    const num = [100, 200, 300, 400, 500];
    
    //map()
    num.map(function(el){
        document.write(el);
    });

    //map(element, index, array)
    num.map(function(element, index, array){
        document.write(element);
        document.write(index);
        document.write(array);
    });
}

11. 배열의 데이터를 펼침 연산자를 통해 불러옵니다.

{
    const num = [100, 200, 300, 400, 500];
    
    document.write(num);
    document.write(num[0], num[1], num[2], num[3], num[4]);
    document.write(...num);
}

12. 배열의 데이터를 배열구조 할당을 통해 불러옵니다.

{
    let a, b, c;

    [a, b, c] = [100, 200, "javascript"];

    document.write(a);
    document.write(b);
    document.write(c);
}

18. 객체 데이터를 hasOwnProperty()을 통해 불러옵니다.

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }

    document.write(obj.hasOwnProperty('a'));
    document.write(obj.hasOwnProperty('b'));
    document.write(obj.hasOwnProperty('c'));
    document.write(obj.hasOwnProperty('d'));
    document.write('a' in obj);
    document.write('b' in obj);
    document.write('c' in obj);
    document.write('d' in obj);
}

19. 객체 데이터를 펼침연산자(복사)를 통해 불러옵니다.

21. 객체 데이터를 펼침연산자(결합)를 통해 불러옵니다.
