01.

이미지/텍스트 유형 : 이미지텍스트01

난이도 쉬움

소스 다운로드

완성 소스 다운받기

소개

안녕하세요!. 웹스토리보이입니다. 이번 주제는 이미지 텍스트 유형을 알아보겠습니다. 이미지 텍스트 유형은 사이트 제작에 있어 많이 사용되는 유형입니다. 이미지와 텍스트가 결합된 형태로 기본 원리는 다른 유형과 동일합니다. 3개의 박스를 만들고 하나만 텍스트 나머지는 이미지를 넣는 유형입니다. 어렵지 않으니 바로 시작하겠습니다.

이론

우리가 이번에 만들어 볼 유형은 이런 디자인입니다. 오른쪽에 이미지가 있고 왼쪽에는 제목과 설명이 있습니다. 우선 imgText01.html 페이지를 만들고, !를 치고 tab을 누르겠습니다. 언어를 ko로 변경하고 title은 이미지 텍스트 유형01로 변경하겠습니다. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>이미지 텍스트 유형01</title>
    <style>
        * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }
        .container {
            width: 1160px;
            margin: 0 auto;
            padding: 0 20px;
            background: rgba(0,0,0,0.3);
            height: 200px;
        }
        .imgText-wrap {
            padding: 120px 0;
            background: rgba(0,0,0,0.1);
        }
    </style>
</head>
<body>
    <section class="imgText-wrap">
        <div class="container">

        </div>
    </section>
</body>
</html>

문서 세팅1을 완료하고, section 태그를 이용하여, 콘텐츠를 설정합니다. 시멘틱 태그2를 이용하여 카드 영역임을 표현합니다. 클래스 이름은 imgText-wrap이라고 설정하겠습니다.

가운데 영역을 설정하기 위해서 container를 클래스를 만들고, width:1160px, padding: 0 20px;을 설정합니다. 블록구조를 가운데 오기3 위해서 margin: 0 auto를 설정하고, 영역 확인을 위해 백그라운드를 넣어줍니다.

head 태그 안에 style 스타일을 선언하고, CSS 초기화4 작업을 하겠습니다. 전체(*)선택자를 이용하여 마진, 패딩, 박스 사이즈를 초기화하겠습니다.

여기까지 결과 확인하기 (imgText_result01_01.html)

결과 처럼 잘 나왔다면, 다음 부분 진행할께요

<div class="imgText__inner">
    <div></div>
    <div></div>
    <div></div>
</div>

imgText__inner 클래스를 추가하고, 자식으로 박스 영역 3개를 만들겠습니다. 첫 번째 박스에는 제목과 설명, 나머지 박스에는 이미지가 추가 될 것입니다. 먼저 3개의 박스 레이아웃 부터 작업하겠습니다. 

.imgText__inner {
    display: flex;
    justify-content: space-between;
}
.imgText__inner > div {
    width: 32%;
    height: 500px;
    background: rgba(0,0,0,0.2);
}

레이아웃 구조는 float, flex, grid 중에 flex5를 사용할 것입니다. 부모 박스에 display: flex;를 선언하고, 자식 박스 정렬을 위해 justify-content: space-between; 설정하겠습니다. 그럼 양쪽 정렬이 되면서 가운데 정렬이 되고, 자식 박스는 width: 32%;를 설정하고, 가상으로 높이 값과 백그라운드를 주겠습니다.

<div class="imgText__inner">
    <div>
        <span>이미지 텍스트 유형01</span>
        <h3>웹스토리보이 사이트</h3>
        <p>
            웹디자이너, 웹 퍼블리셔, 프론트앤드 개발자를 위한 강의 사이트입니다.<br>
            Gmarket Sans Light 22px 150% #67778A 
        </p>
        <a href="#">더 많은 유형 살펴보기</a>
    </div>
    <div></div>
    <div></div>
</div>

첫 번째 박스에 제목, 설명, 버튼 등을 추가하겠습니다. 

.imgText__inner {
    display: flex;
    justify-content: space-between;
    font-family: "GmarketSans";
}
.imgText__inner > div {
    width: 32%;
    height: 500px;
    background: rgba(0,0,0,0.2);
}
.imgText__inner > div:nth-child(1) {
    padding-right: 20px;
}
.imgText__inner span {
    font-size: 16px;
    font-weight: 400;
    margin-bottom: 20px;
    text-decoration: underline;
    text-underline-position: under;
    display: block;
}
.imgText__inner h3 {
    font-size: 50px;
    font-weight: 300;
    margin-bottom: 35px;
}
.imgText__inner p {
    font-size: 18px;
    font-weight: 300;
    line-height: 1.5;
    margin-bottom: 30px;
}
.imgText__inner a {
    font-size: 18px;
    text-decoration: none;
    background: #000;
    color: #fff;
    font-weight: 300;
    padding: 14px 24px;
}

우선 지마켓 웹 폰트6를 추가하겠습니다. 

<link href="https://webfontworld.github.io/gmarket/GmarketSans.css" rel="stylesheet">

font-family: "GmarketSans";를 설정하고, 제목, 설명, 버튼에 해당 속성을 하나씩 설정하겠습니다.

첫 번째 박스한테 padding-right: 20px;을 주어, 여백을 더 확보하겠습니다. 나머지 세부적인 부분은 소스를 참고해서 하나씩 넣어주세요!
