배너 유형 : 배너 유형01

난이도 쉬움

소스 다운로드

소개

안녕하세요!. 웹스토리보이입니다. 이번 주제는 배너유형입니다. 구조는 아주 단순하기 때문에 쉽게 따라 할 수 있습니다. 그럼 바로 시작해보겠습니다.

이론

우리가 이번에 만들어 볼 유형은 이런 디자인입니다. 배경에 이미지가 있고, 가운데 영역에 텍스트가 있는 구조입니다. 우선 banner01.html 페이지를 만들고, !를 치고 tab을 누르겠습니다. 언어를 ko로 변경하고 title은 배너 유형01로 변경하겠습니다. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>배너 유형01</title>
</head>
<body>
    <section class="banner-wrap">
        
    </section>
</body>
</html>

문서 세팅1을 완료하고, section 태그를 이용하여, 콘텐츠를 설정합니다. 시멘틱 태그2를 이용하여 배너 영역임을 표현합니다. 클래스 이름은 banner-wrap이라고 설정하겠습니다.

이제 이미지를 표현해야 합니다. 이미지 표현 방법3 중 의미가 없는 이미지기 때문에 백그라운드로 표현하겠습니다. 이미지는 피그마를 통해 가져오셔도 되고, 다운을 받으셔도 됩니다. 

* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

head 태그 안에 style 스타일을 선언하고, CSS 초기화4 작업을 하겠습니다. 이렇게 하면 쓸데없는 여백이 사라지겠죠!. 이제 이미지를 넣겠습니다. 

.banner-wrap {
    background-image: url(../../assets/img/banner_bg01@2x.jpg);
    background-position: center center;
    background-size: cover;
    background-repeat: no-repeat;
    height: 500px;
}

백그라운드 이미지5를 상대주소6를 기준으로 선언하고, 백그라운드 위치 값을 X축, Y축을 가운데로 설정합니다. 이미지를 화면에 맞추기 위해, 사이즈를 cover로 선언했습니다. 이렇게 하면 영역이 없기 때문에 이미지는 나오지 않습니다. 그래서 가상으로 height 값을 설정하여, 영역을 잡아주겠습니다.

여기까지 잘 따라왔다면, 다음을 진행해보겠습니다. banner__inner div를 만들고, 제목 및 글 등을 설정하겠습니다. 

<div class="banner__inner">
    <h3>유튜버 웹보이</h3>
    <p>
        더 다양한 강의는 유튜브를 통해 제공하고 있습니다.
        <a href="#">youtube.com/c/Webstoryboy</a>
    </p>
    <span>배너 영역01</span>
</div>

클래스 이름도 추가하여, 좀 더 명확하게 해주겠습니다. 

<div class="banner__inner">
    <h3 class="banner__title">유튜버 웹보이</h3>
    <p class="banner__desc">
        더 다양한 강의는 유튜브를 통해 제공하고 있습니다.
        <a href="#">youtube.com/c/Webstoryboy</a>
    </p>
    <span class="banner__sub">배너 영역01</span>
</div>

웹 폰트7를 사용하기 위해, 폰트 파일도 연동하겠습니다. 

<link href="https://webfontworld.github.io/gmarket/GmarketSans.css" rel="stylesheet">

각각 속성에 맞는 폰트 사이즈, 줄 간격, 폰트 두께, 여백 등을 설정해 주겠습니다. 

.banner__inner {
    text-align: center;
    padding: 120px 0;
    color: #fff;
}
.banner__title {
    font-size: 50px;
    font-weight: 300;
    margin-bottom: 40px;
}
.banner__desc {
    font-size: 24px;
    line-height: 1.5;
    font-weight: 300;
    margin-bottom: 70px;
}
.banner__desc a {
    display: block;
    color: #fff;
    text-decoration: none;
}
.banner__sub {
    font-size: 16px;
    font-weight: 300;
    text-decoration: underline;
    text-underline-position: under;
}

.banner__desc a 요소는 인라인 구조이기 때문에 가로로 정렬이 됩니다. 그래서 일시적으로 성질을 블록 구조로 변경하여 줄바꿈을 하였습니다. 이 부분은 블록구조/인라인구조8 개념을 정확히 알아야 합니다. 아주 중요하니까 기억이 안나면 한번 살펴 보고 가세요~

text-decoration을 설정하여 밑줄을 표시하였습니다. 하지만 밑줄이 글씨랑 너무 붙어서 이상하게 보입니다. text-underline-position: under을 주면 조금 아래로 설정할 수 있습니다. 하지만 이 속성은 아직 모든 브라우저에서 적용되지 않는 단점이 있습니다. 만약 예전 브라우저까지 맞추어야 하는 상황이라면 border-bottom: 1px solid #fff 속성을 사용할 수 있습니다.

안에 내용물들이 채워졌으니, 처음에 가상으로 주었던 height:500px 값은 지워주겠습니다.

이제 마지막으로 웹 표준을 준수하면서, 정리하겠습니다. 

<section class="banner-wrap gmarket section">
    <h2 class="blind">배너 제목 영역</h2> 
    <div class="banner__inner">
        <h3 class="banner__title">유튜버 웹보이</h3>
        <p class="banner__desc">
            더 다양한 강의는 유튜브를 통해 제공하고 있습니다.
            <a href="#" target="https://youtube.com/c/Webstoryboy">youtube.com/c/Webstoryboy</a>
        </p>
        <span class="banner__sub">배너 영역01</span>
    </div>
</section>

.section {
    padding: 120px 0;
}
.blind {
    position: absolute;
    clip: rect(0 0 0 0);
    width: 1px;
    height: 1px;
    margin: -1px;
    overflow: hidden
}

section 태그를 만들어 패딩값을 여기로 넣어주었습니다. 섹션마다 해당 타이틀이 있어야 하기 때문에 제목을 넣어주었습니다. 하지만 제목은 디자인 요소로 필요하지 않기 때문에, 우리 눈에는 보이지 않도록 블라인드 효과9를 주었습니다. 웹 접근성10을 위해서 넣어주었습니다. 귀찮아도 이런 부분을 준수해야 웹 표준 사이트가 되겠죠!

마지막으로 링크를 클릭하면 새로운 페이지에서 열리도록 target 설정을 하였습니다.

지금까지 배너유형01을 작업하였습니다. 그렇게 어려운 부분은 없다고 보이는데, 그래도 궁금한 점이나 안되는 부분은 마지막 완료 코드 확인해 보시고, 그래도 안되면 discord나 댓글로 문의주시면 도와드리겠습니다. 수고하셨습니다. 다음 유형에서 뵙겠습니다. 😻
