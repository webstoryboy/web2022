푸터 유형 : 푸터01

난이도 쉬움

소스 다운로드

완성 소스 다운받기

소개

안녕하세요!. 웹스토리보이입니다. 이번 주제는 이미지 텍스트 유형을 알아보겠습니다. 이미지 텍스트 유형은 사이트 제작에 있어 많이 사용되는 유형입니다. 이미지와 텍스트가 결합된 형태로 기본 원리는 다른 유형과 동일합니다. 3개의 박스를 만들고 하나만 텍스트 나머지는 이미지를 넣는 유형입니다. 어렵지 않으니 바로 시작하겠습니다.

이론

우리가 이번에 만들어 볼 유형은 이런 디자인입니다. 오른쪽에 이미지가 있고 왼쪽에는 제목과 설명이 있습니다. 우선 imgText01.html 페이지를 만들고, !를 치고 tab을 누르겠습니다. 언어를 ko로 변경하고 title은 이미지 텍스트 유형01로 변경하겠습니다. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>이미지 텍스트 유형</title>
    <style>
        * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }
        .imgText-wrap {
            width: 100%;
            background: rgba(0,0,0,0.2);
        }
        .section {
            padding: 120px 0;
            text-align: center;
        }
        .container {
            width: 1160px;
            margin: 0 auto;
            padding: 0 20px;
            background: rgba(0,0,0,0.3);
        }
        .imgText-inner {
            height: 200px;
        }
    </style>
</head>
<body>
    <section class="imgText-wrap section gmark">
        <div class="container">
            <div class="imgText-inner">
            </div>
        </div>
    </section>
</body>
</html>
여기까지 결과01 확인하기

전체 영역을 표현 할 수 있는 .imgText-wrap 클래스를 만들고 자식으로 가운데 영역을 표현할 .container를 만들겠습니다. 전체 선택자(*)를 이용하여 기본값을 초기하고, 박스 사이즈 값도 초기화 하겠습니다. .imgText-wrap 클래스에는 width:100%와 백그라운드 색으로 영역으로 표시하겠습니다. .section 클래스도 추가하여 padding: 120px 0 값을 추가하겠습니다.

.container 클래스를 추가하여, 가운데로 올 수 있도록 설정하겠습니다. width: 1160px;, margin: 0 auto;, padding: 0 20px;, background: rgba(0,0,0,0.3);를 추가하여 가운데로 오게 하고 영역을 확인하기 위해 백그라운드 색도 넣어주겠습니다.

.imgText-inner 클래스를 만들고 가상으로 height: 200px만 넣어주겠습니다. 이렇게 해주면 결과01처럼 나옵니다. 여기까지 잘 나왔다면, 다음 부분 진행하겠습니다. 

<div class="imgText-inner">
    <div class="imgText"></div>
    <div class="imgText"></div>
    <div class="imgText"></div>
</div>

박스를 3개더 만들고 .imgText 클래스 이름을 추가하였습니다. 

.imgText-inner {
    display: flex;
    justify-content: space-between;
}
.imgText-inner .imgText {
    width: 32%;
    height: 512px;
    background: rgba(0,0,0,0.4)
}

display: flex;와 justify-content: space-between;를 추가하고, 자식 박스 .imgText 클래스에 width, height을 설정하고, 영역 확인을 위해 background를 설정합니다.
