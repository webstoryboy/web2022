소개

안녕하세요!. 웹스토리보이입니다. 사이트 만들기 위한 가장 기본적인 레이아웃 유형을 살펴보겠습니다. 레이아웃은 사이트를 만들기 위한 가장 기본적인 단계입니다. 사이트는 레이아웃 속에 레이아웃으로 구조화 되어 있기 때문에 레이아웃을 이해한다면, 사이트 제작이 수월해집니다. 레이아웃을 만드는 방법은 여러가지로 나눌 수 있습니다. table 방식, float 방식, flex 방식, grid 방식으로 나눌 수 있습니다.

table 방식은 2000년 대 이전 방식으로 웹 표준을 준수하지 않은 방법으로 현재는 사용하지 않습니다. float 방식은 2020년 대 이전 방식으로 테이블 방식 대신 이 방법으로 주로 구현되어 왔습니다. 레이아웃을 위한 CSS 속성은 정확하게 존재하지 않았습니다. 그래서 float을 이용하게 되었지만 버그도 발생하고, 레이아웃을 구조화하는데 어려움이 있었습니다. 그래서 현재는 flex 방식과 grid 방식을 사용합니다. 플렉스와 그리드는 레이아웃을 위한 CSS의 새로운 속성이라고 생각하시면 됩니다. 하지만 현재 실무에서는 아직도 float 방식을 많이 사용합니다. 기존에 만들어졌던 사이트들이 float 방식으로 되어 있기 때문에 여러분들은 float 방식도 공부하셔야 합니다. 그래서 저희는 float, flex, grid 세가지 방식으로 레이아웃을 작업할 것입니다. 그럼 준비 되셨죠? 시작해 볼까요! 🤪

이론

이렇게 생긴 디자인을 작업할 것입니다. 레이아웃 유형에서 가장 많이 사용하는 구조로 여러분들도 많이 보셨을겁니다. 레이아웃 방식은 크게 세가지(float 방식, flex 방식, grid 방식)가 있다고 했습니다. 이 방식을 하나씩 작업해보겠습니다. 즉 똑같은 레이아웃을 3가지 방법으로 작업해 볼 것입니다. 그럼 float 방식을 먼저 해보겠습니다.

가장 먼저 VScode에서 webstandard 폴더를 만들고 01_layout01.html 파일을 만들겠습니다.

!를 치고 tab을 누르겠습니다. 언어를 ko로 변경하고 title은 레이아웃 유형01로 변경하겠습니다.

레이아웃 작업을 할 때는 블록 요소1로 레이아웃을 작업합니다. 블록은 하나의 박스라고 생각하면 됩니다. 그림과 같은 레이아웃은 제일 큰 박스 안에 5개의 박스로 구성된 구조로 보시면 됩니다. 결국 레이아웃은 퍼블조각 맞추기라고 생각하시면 쉽니다. 그럼 하나식 따라해보겠습니다.

1. float으로 레이아웃 구성하기

<div>
    <div></div>
    <div></div>
    <div></div>
    <div></div>
    <div></div>
</div>

HTML 구조는 이렇게 간단하게 끝납니다. 하지만 똑같은 div 태그로 이루어여 있기 때문에 구별을 해주어야 합니다. 그래서 이름을 지워주겠습니다. 이렇게 이름을 지워주고 선택할 수 있습니다. 이것을 선택자(selector)2라고 합니다. 저는 ID 선택자를 이용하여 이름을 지워주었습니다. 

<div id="wrap">
    <div id="header"></div>
    <div id="nav"></div>
    <div id="aside"></div>
    <div id="section"></div>
    <div id="footer"></div>
</div>

다음은 CSS를 통해서 영역을 잡고, 배경색을 넣어보겠습니다. 

<style>
    #wrap {
        width: 1200px;
    }
    #header {
        width: 1200px;
        height: 100px;
        background-color: #FFE0B2;
    }
    #nav {
        width: 1200px;
        height: 100px;
        background-color: #FFCC80;
    }
    #aside {
        width: 400px;
        height: 780px;
        background-color: #FFB74D;
    }
    #section {
        width: 800px;
        height: 780px;
        background-color: #FFA726;
    }
    #footer {
        width: 1200px;
        height: 100px;
        background-color: #FF9800;
    }
</style>

이렇게 width값과 height 값을 넣어주고, background-color 색을 넣어주면 영역을 확인할 수 있습니다. 한번 확인해 볼까요?
